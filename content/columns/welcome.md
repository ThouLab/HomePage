# ThouLab をはじめます

> 静かに深く、学びをひらく。

ThouLab は **学び・実験・共有** を循環させるための小さなラボです。  
このサイトは GitHub Pages で公開できる静的サイトとして設計し、コンテンツは `content/` 配下で管理します。

## 何をするのか

- 問いを置く（目的・前提・評価軸を言語化）
- 小さく試す（プロトタイプ / ハンズオン）
- 言葉で残す（コラム・資料・テンプレート）
- 改善して次へ（フィードバックで更新）

## サイトの作り

このリポジトリの要点は次のとおりです。

1. `partials/` … ヘッダー / フッター
2. `content/` … JSON / Markdown（増やしやすい）
3. `assets/` … CSS / JS / 画像

### コードブロック例

```js
// content/columns/index.json にメタ情報を追加し、
// content/columns/<slug>.md を置くだけで記事が増えます。
```
